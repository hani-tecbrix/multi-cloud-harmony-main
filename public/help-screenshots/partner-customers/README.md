# Partner Customers Help Guide Screenshots

This directory contains screenshots for the Partner Customers help guide.

4. **customer-type-selection.png** - Add Customer panel showing New Customer vs Existing Customer options
5. **cloud-provider-selection.png** - Cloud provider selection cards for AWS, Azure, and GCP
6. **customer-information-form.png** - Form fields for entering customer information
7. **tenant-detection-before.png** - Cloud Tenant Detection section before running detection
8. **tenant-detection-loading.png** - Tenant detection in progress with loading indicator
9. **tenant-detection-results.png** - Detected cloud tenants with account details and costs
10. **service-selection.png** - Additional cloud services selection interface
11. **add-customer-success.png** - Success notification after adding a customer
12. **customers-table.png** - Customers table showing customer data with all columns
13. **search-functionality.png** - Search bar with search term and filtered results
14. **pagination-controls.png** - Pagination controls showing page numbers and navigation
15. **customer-detail-view.png** - Full customer detail view with overview and metrics
16. **billing-information.png** - Billing Information card with payment details
17. **subscription-details.png** - Subscription Details table showing all active subscriptions

## Instructions for Taking Screenshots

1. Start the development server: `npm run dev`
2. Log in as a partner user
3. Navigate to `/partner/customers`
4. Take screenshots at each stage of the workflow
5. Save screenshots with the exact filenames listed above
6. Use PNG format for all screenshots
